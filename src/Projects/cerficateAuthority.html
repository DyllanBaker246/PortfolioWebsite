Overview

For part of my Cryptography final (CPSC 352), I created a certificate authority which managed and provided secure HTTPS communication across various web servers.

This project gave me hands on experience with many important cyrptographic systems such as SSL/TLS, X.509 certificates, certificate chains, and public key infustructure. I also got hands on practice with setting up servers such as Apache, Microsoft IIS, and Node.js.

Please see the documentation and video demo for more information.
